🏷 Semantic HTML Tags <semantic>

Semantic tags give meaning to content. They help browsers, search engines, and screen readers understand page structure.

📰 Article: Why Semantics Matter

Using tags like <header>, <nav>, <main>, <aside>, and <footer> tells browsers exactly what each section means — not just how it looks.

What is <header>?

The <header> tag defines introductory content — usually a logo, page title, and navigation. It can appear inside <body>, <section>, or <article>.

What is <aside>?

The <aside> tag holds content that is indirectly related to the main content — like sidebars, pull quotes, or advertisements.

What is <figure> & <figcaption>?

Used together to mark up self-contained media (images, diagrams, code) and their caption.

📄 3 Types of CSS CSS

Type Where Written Syntax Example Best For
Inline Inside HTML tag style="color:red;" Quick single overrides
Internal In <head> → <style> p { color: blue; } Single-page styling
External Separate .css file <link rel="stylesheet"> Multi-page projects

Inline CSS — applied directly on tags below:

This text is red & bold | Amber background | Teal underline | Big purple text

Internal CSS — styles defined in <head> <style>:

/* Internal CSS — inside <style> in <head> */ body { background-color: #fdf6ee; /* hex */ color: hsl(28, 60%, 15%); /* hsl */ } header { background-color: rgba(45,32,19,1); /* rgba */ border-bottom: 3px solid coral; /* named */ }

External CSS — linked via <link> tag:

<!-- In HTML <head> --> <link rel="stylesheet" href="style.css" /> /* In style.css file */ .highlight-primary { background-color: hsl(28, 100%, 95%); color: hsl(28, 80%, 30%); border-radius: 4px; }
