🏷 Semantic HTML Tags <semantic>

Semantic tags give meaning to content. They help browsers, search engines, and screen readers understand page structure.

📰 Article: Why Semantics Matter

Using tags like <header>, <nav>, <main>, <aside>, and <footer> tells browsers exactly what each section means — not just how it looks.

What is <header>?

The <header> tag defines introductory content — usually a logo, page title, and navigation. It can appear inside <body>, <section>, or <article>.

What is <aside>?

The <aside> tag holds content that is indirectly related to the main content — like sidebars, pull quotes, or advertisements.

What is <figure> & <figcaption>?

Used together to mark up self-contained media (images, diagrams, code) and their caption.
